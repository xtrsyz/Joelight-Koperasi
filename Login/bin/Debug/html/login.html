Tampilan Login Aplikasi

Tampilan login ini berfungsi sebagai otentikasi user ketika akan menggunakan aplikasi.
Untuk proses otentikasi ada 3 pilihan yaitu masuk sebagai Admin, User
Jika Anda ingin masuk sebagai pengguna biasa dan ingin melakukan konsultasi menggunakan aplikasi ini maka Anda bisa menggunakan pilihan untuk masuk sebagai User dengan cara :

  • Username di isi dengan Umum dan Password di isi dengan umum
  • Setelah itu klik tombol Ok
  • Anda telah masuk sebagai User Status hak akses anda bisa di lihat pada bagian bawah dari pada menu

Sedangkan apabila Anda ingin melakukan perubahan terhadap basis data dalam aplikasi, maka Anda harus masuk sebagai Admin :

  • Isikan username dan password dari Admin
  • Setelah itu klik tombol Masuk
  • Anda telah masuk sebagai Admin
