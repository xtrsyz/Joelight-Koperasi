Tampilan Halaman Profil dan Mengganti Password

Tampilan halaman profil halaman ini berfungsi untuk mengetahui profil anggota koperasi atau data-data pribadi anggota koperasi dan halaman ini juga berfungsi untuk mengubah password untuk Admin dan User/Karyawan
Cara untuk melihat profil yaitu :

  • Tekan Menu
  • Pilih Master Anggota
  • Pilih cari Anggota
  • Masukkan NIK atau Nama Anggota Koperasi tersebut lalu klik tombol cari

Sedangkan apabila Anda ingin melakukan perubahan password caranya sebagai berikut :

  • Tekan Menu
  • Pilih Master Anggota
  • Pilih Lihat Profil di halaman profil terdapat form mengganti password
  • masukkan password lama Anda
  • Masukkan password baru Anda
  • Ulangi password baru Anda lalu klik Ganti Password
