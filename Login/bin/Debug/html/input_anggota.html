Tampilan Halaman Memasukkan, Mengedit dan Menghapus Data Anggota Koperasi

Halaman memasukkan data anggota merupakan halaman untuk memasukkan data-data pribadi untuk menjadi anggota koperasi yang baru dan bisa mengedit data-data anggota koperasi yang lama dan sudah tersimpan serta dapat menghapusnya. Proses ini hanya bisa dilakukan oleh admin saja.
Cara untuk memasukkan data anggota koperasi yaitu :

  • pilih Menu
  • Pilih Master Anggota
  • Pilih input Anggota
  • Masukkan Data Anggota lalu klik tombol Simpan

Sedangkan apabila Admin ingin mengedit dan menghapus data anggota caranya sebagai berikut :

  • pilih Menu
  • Pilih Master Anggota
  • Pilih input anggota
  • Masukkan NIK lalu klik tombol lihat
  • pilih tombol update atau hapus sesuai dengan kebutuhan admin
